# Polygon Editor

Тестовое задание для frontend-разработчика.

## Что реализовано

- Генерация случайных полигонов от 3 до 7 вершин
- Поддержка выпуклых и не выпуклых фигур
- Отрисовка на `canvas`
- Выбор полигона по клику
- Перетаскивание целиком с ограничением по границам холста
- Запрет наложения полигонов друг на друга
- Удаление выбранного полигона и удаление всех полигонов
- Undo / Redo для:
  - добавления
  - удаления
  - удаления всех
  - перемещения
  - изменения цвета
  - импорта сцены
- Кнопки и горячие клавиши:
  - `Ctrl+Z` – Undo
  - `Ctrl+Y` и `Ctrl+Shift+Z` – Redo
  - `Delete` – удалить выбранный полигон
- Информационная панель
- Toast-сообщения
- Лёгкая анимация появления нового полигона

## Бонус

- Изменение цвета выбранного полигона
- Экспорт и импорт сцены в JSON

## Стек

- Чистый JavaScript
- Web Components
- Canvas API
- Webpack

## Установка и запуск

```bash
npm install
npm start
```

Приложение откроется на `http://localhost:8080`.

## Production build

```bash
npm run build
```

Сборка появится в папке `dist`.
